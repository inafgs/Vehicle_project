# Vehicle_project
Interactive web app to explore and visualize U.S. car sales data using Python, Plotly and Streamlit.

**U.S. Car Sales Dashboard**
An interactive web application to explore and visualize a dataset of car sales advertisements in the United States.
Features

Odometer Histogram – shows the distribution of mileage across listed vehicles
Scatter Plot – explores the relationship between mileage and sale price

**Technologies**

Python
Pandas
Plotly
Streamlit

**How to Run Locally**

1. Clone this repository
2. Create and activate a virtual environment
3. Install dependencies:
pip install -r requirements.txt

4. Launch the app:
streamlit run app.py
